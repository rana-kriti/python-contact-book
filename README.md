# python-contact-book

A simple Command-Line Contact Book application built with Python that allows users to store and manage contacts easily.

This project demonstrates basic Python concepts such as:
- Functions
- Dictionaries
- Loops
- Conditional statements
- User input handling

## Features
✔ Add a new contact
✔ View all saved contacts
✔ Search for a contact by name
✔ Delete a contact
✔ Simple command-line interface

## Menu Options
The program provides the following options:
1. Add Contact – Add a new contact with name, phone number, and email
2. View Contacts – Display all saved contacts
3. Search Contact – Find a contact by name
4. Delete Contact – Remove a contact from the list
5. Exit – Close the program
